# 正则表达式
---

## RegExp

JavaScript中正则表达式通过内置的RegExp来实现。

```javascript
// 创建正则表达式语法
regexp = new RegExp('pattern', 'flags');  // 允许动态构建(使用变量), 需要注意转义
regexp = /pattern/[flags];  // 不允许动态构建

/* flags值:
i  搜索时不区分大小写: A 和 a 没有区别
g  搜索时会查找所有的匹配项, 而不只是第一个
m  多行模式, ^和$将匹配每行的开始和结束
u  开启完整的unicode支持, 该修饰符能够修正对于代理对的处理
y  粘滞模式
*/

// regexp属性和方法
regexp.test(str);  // 在str中匹配, 返回true/false
regexp.exec(str);  // 返回str中所有匹配的子串
regexp.lastIndex = num;  // 设置匹配结束位置, 需要使用y模式才会生效 
```

## 正则规则

正则表达式语法:

```shell
## 单个字符类
x   单个字符元素, 如: 1, s 等
.   任意字符, 除\n外
\d  数字 [0-9]
\D  非数字 [^0-9]
\s  空格 [\t\n\f\r ]
\S  非空格 [^\t\n\f\r ]
\w  字母, 数字, 下下划线 [0-9A-Za-z_]
\W  非\w [^0-9A-Za-z_]

## 复合体
xy  表示x后紧跟y
x|y 表示x或y, x优先

## 空字符串
^   文本或行的开始, 如: ^Ha
$   文本或行的结尾, 如: Bye$
\A  文本开始
\z  文本结束
\b  边界词, 如: \bShell\b
\B  非边界词

## 转义
[ \ ^ $ . | ? * + ( )  特殊字符前需要加 \ 进行转义
\f   换页符
\t   制表符tab
\n   换行符
\r   回车符
\v   垂直制表符
\123 八进制(最多三位)
\x7F 十六进制(两位)

## 集合
[xyz]  集合中的任意一个字符, 集合不需要转义的字符[-().^+]
[^xyz] 除集合外的任意一个字符
[a-z]  范围中的任意一个字符

## 重复
最多匹配模式
x*  0或多个x
x+  1或多个x
x?  0或1个x
x{n, m}  x的数量在[n, m]
x{n,}    x的数量在[n, +&)
x{n}     n个x

最少匹配模式, 就是在最多匹配模式后加 ？
x+?  1或多个x
...

## 分组
(xy)    捕获组, 将xy看成一个整体
(?:xy)  排除捕获组
(?<name>re)  命名捕获组, JS中的模式, 将该组匹配结果分配给name, 以类map形式返回
(?P<name>re) 命名捕获组, go, rust中的模式
x(?=y)  当x后为y时匹配x
x(?!y)  当x后不是y时匹配x
(?<=y)x 当x前面是y时匹配x
(?<!y)x 当x前面不是y时匹配x
```
